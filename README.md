# 🎮 Hangman Game

## 📌 Project Overview
This is a simple text-based Hangman Game developed in Python as part of the **CodeAlpha Python Programming Internship**.

The player has to guess a randomly selected word one letter at a time. The player wins by guessing all letters before making six incorrect guesses.

## ✨ Features
- Randomly selects a word from a predefined list
- Maximum of 6 incorrect guesses
- Displays guessed letters
- Prevents duplicate guesses
- Simple console-based interface

## 🛠️ Technologies Used
- Python 3

## 📚 Python Concepts Used
- Random Module
- Lists
- Strings
- While Loop
- If-Else Statements
- User Input
- Functions

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Mandeepsharma123/CodeAlpha_HangmanGame.git
```

2. Open the project folder

3. Run the program

```bash
python hangman_game.py
```
